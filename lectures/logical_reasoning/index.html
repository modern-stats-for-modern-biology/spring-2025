Knowledge-based AI

  • Logic (e.g., propositional logic or first-order logic) is a language used to represent knowledge and to reason (draw conclusions, make inferences) based on this knowledge

  • Logic is used in a knowledge-based AI agent to represent knowledge and to reason (draw conclusions, make inferences) based on this knowledge

  • In deductive logic the goal is to determine whether an argument is valid (the premises entail the conclusion), using some inference algorithm or rules

  • In inductive logic the goal is to determine whether an argument is strong (the premises partially entail the conclusion), using some inference algorithm or rules

Deductive inference: Propositional logic

  • The goal is to prove that a conclusion/hypothesis/query is true given some premises/evidence/knowledge known or assumed to be true

  • \(E \models C\)

  • \(P(C \mid E) = 1\)

Propositional symbols/variables

In propositional logic, the terms propositional symbol and propositional variable are often used interchangeably to denote atomic propositions—statements that are either true or false and cannot be broken down into simpler statements. These atomic propositions serve as the foundational building blocks for more complex, compound propositions formed using logical connectives/boolean operators such as AND, OR, and NOT.

Standard linear regression likelihood

  • The regression likelihood \(p(y \mid x, \theta)\) is:

\[p(y \mid \mu, \sigma, M_N) \quad \mu = \beta_0 + \beta_1 \cdot x\]

\[Y \sim N(\mu, \sigma) \quad \mu = \beta_0 + \beta_1 \cdot x\]

\[p(y \mid \mu, \sigma, M_N) = \frac{1}{\sigma \sqrt{2\pi}} \cdot e^{-\frac{1}{2} \left( \frac{y - \mu}{\sigma} \right)^2}\]

\[p(y \mid x, \beta_0, \beta_1, \sigma, M_{L+E_N}) = \frac{1}{\sigma \sqrt{2\pi}} \cdot e^{-\frac{1}{2} \left( \frac{y - (\beta_0 + \beta_1 \cdot x)}{\sigma} \right)^2}\]

Process model and statistical inference

  • Given a process model (a.k.a. data model, generative model, or likelihood):

\[p(x, y \mid \theta)\]

  • The corresponding inference model is:

\[p(\theta \mid x, y) = \frac{p(\theta) \cdot p(x, y \mid \theta)}{p(x, y)}\]

  • \(p(\theta \mid x, y)\) is called the posterior distribution

  • \(p(\theta)\) is called the prior distribution

  • \(p(x, y \mid \theta)\) is called the likelihood

  • \(p(x, y)\) is a normalizing constant a.k.a. the marginal likelihood \(= \sum_\theta p(x, y \mid \theta)\)

\[p(\theta \mid x, y) \propto p(\theta) \cdot p(x, y \mid \theta)\]

Bayesian inference using the linear model

  • The inference model:

\[p(\theta \mid x, y) \propto p(\theta) \cdot p(x, y \mid \theta)\]

  • Assuming a standard linear regression model:

\[p(\beta_0, \beta_1, \sigma \mid x, y) \propto p(\beta_0, \beta_1, \sigma) \cdot p(y \mid x, \beta_0, \beta_1, \sigma)\]

  • Assuming \(p(\beta_0, \beta_1, \sigma) = p(\beta_0) \cdot p(\beta_1) \cdot p(\sigma)\):

\[p(\beta_0, \beta_1, \sigma \mid x, y) \propto p(\beta_0) \cdot p(\beta_1) \cdot p(\sigma) \cdot p(y \mid x, \beta_0, \beta_1, \sigma)\]

prior = d?(b0) * d?(b1) * d?(sigma)

likelihood = dnorm(Y, mean = b0 + b1 * X, sd = sigma)

posterior.unnormalized = prior * likelihood

posterior = posterior.unnormalized / sum(posterior.unnormalized)

Prior probability distribution

  • Assuming the prior probability distribution of parameter \(\beta_0\) is:

\[ p(\beta_0, \text{min} = 0, \text{max} = 15, M_U) \quad \beta_0 \sim U(\text{min} = 0, \text{max} = 15) \]

p = dunif(b0, min = 0, max = 15)
  • Assuming the prior probability distribution of parameter \(\beta_1\) is:

\[ p(\beta_1, \text{min} = -15, \text{max} = 15, M_U) \quad \beta_1 \sim U(\text{min} = -15, \text{max} = 15) \]

p = dunif(b1, min = -15, max = 15)
  • Assuming the prior probability distribution of parameter \(\sigma\) is:

\[ p(\sigma, \text{min} = 1, \text{max} = 6, M_U) \quad \sigma \sim U(\text{min} = 1, \text{max} = 6) \]

p = dunif(sigma, min = 1, max = 6)

Prior probability distribution

  • The joint prior probability distribution of parameters \(\beta_0\), \(\beta_1\), and \(\sigma\) is:

\[p(\beta_0, \text{min} = 0, \text{max} = 15, M_U) \cdot p(\beta_1, \text{min} = -15, \text{max} = 15, M_U) \cdot p(\sigma, \text{min} = 1, \text{max} = 6, M_U)\]

prior = dunif(b0, min = 0, max = 15) * dunif(b1, min = -15, max = 15) * dunif(sigma, min = 1, max = 6)

Posterior probability distribution

  • Compute posterior probability distribution by multiplying prior and likelihood:
ss %<>%
  mutate(
    data = list(d.obs) # add a list column containing d.obs in each row
  ) %>%
  rowwise() %>%
  mutate(
    prior = ddunif(b0, min = 0, max = 15) * ddunif(b1, min = -15, max = 15) * ddunif(sigma, min = 1, max = 6),
    likelihood = {
      data %>%
        mutate(
          mu = b0 + b1 * X,
          likelihood = ddnorm(Y, mu, sigma)
        ) %>%
        pull(likelihood) %>%
        prod()
    },
    posterior.unnormalized = prior * likelihood
  ) %>%
  ungroup() %>%
  mutate(posterior = posterior.unnormalized / sum(posterior.unnormalized))

    likelihood = {

        mutate(
          mu = b0 + b1 * X,
          likelihood = ddnorm(Y, mu, sigma)
        ) %>%
        pull(likelihood) %>%
        prod()
    },
    posterior.unnormalized = prior * likelihood
  ) %>%
  ungroup() %>%
  mutate(posterior = posterior.unnormalized / sum(posterior.unnormalized))

posterior_predictive <- posterior_draws %>%
  rowwise() %>%
  mutate(
    Y_pred = list(
      d.obs %>%
        mutate(
          Y_sim = b0 + b1 * X + runif(n(), min = -s, max = s)  # predict Y with uniform noise
          # Y_sim = b0 + b1 * X + rnorm(n(), mean = 0, sd = s)  # predict Y with normal noise
        )
    )
  ) %>%
  unnest(Y_pred)
posterior_summary <- posterior_predictive %>%
  group_by(X) %>%
  summarise(
    Y_mean = mean(Y_sim),
    Y_lower = quantile(Y_sim, 0.025),
    Y_upper = quantile(Y_sim, 0.975)
  )

posterior_predictive_diff <- posterior_draws %>%
  rowwise() %>%
  mutate(
    # Simulated predictions for X = 0 and X = 1
    Y_pred_0 = list(b0 + b1 * 0 + runif(1000, min = -s, max = s)),  # For X = 0
    Y_pred_1 = list(b0 + b1 * 1 + runif(1000, min = -s, max = s))   # For X = 1
  ) %>%
  mutate(
    # Compute the mean difference between X = 1 and X = 0 for each posterior draw
    diff = mean(unlist(Y_pred_1) - unlist(Y_pred_0))
  ) %>%
  ungroup()
# Summarize the differences
summary_diff <- posterior_predictive_diff %>%
  summarise(
    point_estimate = mean(diff),           # Posterior mean of the difference
    lower_ci = quantile(diff, 0.025),      # 2.5% credible interval
    upper_ci = quantile(diff, 0.975)       # 97.5% credible interval
  )

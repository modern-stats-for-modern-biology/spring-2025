From Nutella on toast to statistical models

Introduction

Imagine a piece of toast that’s come out of the toaster with a faint grid pattern pressed in by the rack. We’ll treat each little square in that pattern as a possible outcome in our sample space. Next, we spread 1 Kg of Nutella across the toast so that some squares get a thick coating and others barely any at all.

Wherever the Nutella is thick, the probability is high; where it’s thin or absent, the probability is low or zero. But taken together, the total Nutella spread must sum to 1 Kg.

In this notebook, we’ll model several probability distributions/stastistical models as different ways of smearing Nutella across the toast.

Once we have our sample space and probability distribution, we can:

  • Calculate the probability of any proposition/event in the sample space, i.e., any conclusion given the premises/assumptions/evidence

  • Sample from it to generate data in accordance with the model

plot_model <- function(pdf, n = NULL) {
  n_cells <- nrow(pdf)
  
  # Decide style logic based on total number of cells
  if (n_cells < 100) {
    line_color  <- "black"
    line_width  <- 0.5
    text_size   <- 3
  } else if (n_cells < 200) {
    line_color  <- "black"
    line_width  <- 0.3
    text_size   <- 2
  } else if (n_cells < 500) {
    line_color  <- "black"
    line_width  <- 0.2
    text_size   <- NA
  } else {
    line_color  <- NA
    line_width  <- 0
    text_size   <- NA
  }
  
  # If samples are requested, do not add text labels
  if (!is.null(n)) {
    text_size <- NA
  }
  
  # Determine number of distinct X and Y values for axis text sizing
  n_x <- length(unique(pdf$X))
  n_y <- length(unique(pdf$Y))
  
  axis_text_x_size <- if(n_x < 10) { 12 } else if(n_x < 20) { 10 } else if(n_x < 40) { 8 } else { 6 }
  axis_text_y_size <- if(n_y < 10) { 12 } else if(n_y < 20) { 10 } else if(n_y < 40) { 8 } else { 6 }
  
  max_x <- max(pdf$X)
  max_y <- max(pdf$Y)
  
  # Determine the fill scale parameters based on n_cells:
  # For small grids, the range of p is set higher (midpoint and limit are larger)
  # For larger grids, the scale is "compressed" to better reflect small p values.
  if (n_cells < 100) {
    mid_val   <- 0.25
    limit_val <- 1
  } else if (n_cells < 200) {
    mid_val   <- 0.125
    limit_val <- 0.5
  } else if (n_cells < 500) {
    mid_val   <- 0.0625
    limit_val <- 0.25
  } else {
    mid_val   <- 0.03125
    limit_val <- 0.125
  }
  
  fill_scale <- scale_fill_gradientn(
    colors = c("white", "chocolate4", "black"),
    values = c(0, mid_val, 1),
    limits = c(0, limit_val),
    oob = scales::squish
  )
  
  x_scale <- scale_x_continuous(breaks = seq(0, max_x))
  y_scale <- scale_y_continuous(breaks = seq(0, max_y))
  base_theme <- theme(
    panel.grid.major = element_blank(),
    panel.grid.minor = element_blank(),
    axis.text.x = element_text(size = axis_text_x_size, margin = margin(t = 2, b = 0)),
    axis.text.y = element_text(size = axis_text_y_size, margin = margin(r = 2, l = 0))
  )
  
  # Base tile plot
  plt <- gf_tile(
    Y ~ X,
    data = pdf,
    fill = ~ p,
    color = line_color,
    linewidth = line_width
  ) %>%
    gf_refine(
      fill_scale,
      x_scale,
      y_scale,
      base_theme
    ) +
    expand_limits(x = c(0, max_x), y = c(0, max_y)) +
    coord_fixed(expand = FALSE)
  
  # Optionally add text labels if text_size is applicable
  if (!is.na(text_size)) {
    plt <- plt %>% gf_text(label = ~ sprintf("%0.3f", p), size = text_size, alpha = 0.75)
  }
  
  # Optionally overlay random sample points if n is provided
  if (!is.null(n)) {
    plt <- plt %>% gf_point(
      Y ~ X,
      data = pdf %>% slice_sample(n = n, weight_by = p, replace = TRUE),
      size = 4,
      color = "red2",
      alpha = 0.5,
      stroke = 0
    )
  }
  
  return(plt)
}

\[ \begin{aligned} p(x, y \mid M_2) & = p\bigl(x, y \mid (y = 3 + x + e), e \sim \text{Uniform}(\mu = 0, \sigma = 1)\bigr) \\ & = p\bigl(x, y \mid \beta_0 = 3, \beta_1 = 1, \sigma = 1, M_{L+E_\text{Uniform}}\bigr) \\ \\ & M_{L+E_\text{Uniform}}: y = \beta_0 + \beta_1 \cdot x + e \quad e \sim \text{Uniform}(\mu = 0, \sigma) \\ \\ & M_{L+E_\text{Uniform}}: y \sim \text{Uniform} \bigl(\mu = (\beta_0 + \beta_1 \cdot x), \sigma \bigr) \end{aligned} \]

How to fit any dataset with a single parameter

https://arxiv.org/abs/1904.12320

“With four parameters I can fit an elephant and with five I can make him wiggle his trunk.” from a discussion between physicists John von Neumann and Enrico Fermi

Von Neumann’s elephant

\[f(x) = \sin^2 (2^{x\tau} \arcsin \sqrt{\alpha})\]

\(\tau\) is a constant that controls the desired level of accuracy

f <- function(x, tau, alpha) {
  angle <- 2^(x * tau) * asin(sqrt(alpha))
  sin(angle)^2
}

x <- seq(-1, 1, length.out = 1000)

alpha <- 0.5

y <- f(x, alpha = alpha, tau = 3)

tibble(x = x, y = y) %>%
  gf_point(y ~ x) %>%
  gf_labs(x = "x", y = "f(x)")
